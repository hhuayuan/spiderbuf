
# Spiderbuf 爬虫练习网站

Spiderbuf 网络爬虫实战案例示例代码

## 官方网站

https://spiderbuf.cn

[爬虫实战练习](https://spiderbuf.cn/challenges)

专业的爬虫练习靶场。

从基础 HTML 解析到复杂的 JS 逆向渲染，提供全场景数据抓取练习环境。

无需配置环境，打开浏览器即可实战。
        

## 代码运行环境

Ubuntu 20.04.6 LTS \ macOS 15+

Python3.8+
