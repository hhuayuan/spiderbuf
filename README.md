# spiderbuf
Spiderbuf 示例代码

# 官方网站
https://spiderbuf.cn

Spiderbuf 是一个python爬虫学习及练习网站：

保姆式引导关卡 + 免费在线视频教程，从Python环境的搭建到最简单的网页爬取，让零基础的小白也能获得成就感。

在已经入门的基础上强化练习，在矛与盾的攻防中不断提高技术水平，通过大量的模仿练习掌握常见的爬与反爬套路。

以闯关的形式挑战各个关卡任务，验证自身实力的时候到了。

# 代码运行环境
Ubuntu 20.04.6 LTS

Python3.8.10
